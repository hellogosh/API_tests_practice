<B>О проекте</B>

Данный проект содержит автоматические тесты для проверки API по основным CRUD-операциям с ресурсами 

https://dashboard.fast-api.senior-pomidorov.ru/:

-Создание нового элемента.

-Получение списка элементов с проверкой структуры, фильтрации и пагинации.

-Полное обновление элемента по ID.

-Удаление элемента по ID.


<B>Быстрый старт:</B>

Необходимое ПО
Python 3.9+

Виртуальное окружение (рекомендуется)

Установленные зависимости (pytest, requests)

<B>Установка и подготовка</B>

Клонируйте репозиторий:

*bash
git clone https://github.com/hellogosh/API_tests_practice.git*

*cd API_Practice*

<B>Создайте и активируйте виртуальное окружение:</B>

bash

python -m venv venv

source venv/bin/activate   # Linux/macOS

venv\Scripts\activate      # Windows

<B>Установите зависимости:</B>

bash

pip install -r requirements.txt

Настройте файл config/constant.py — укажите BASE_URL и другие параметры для доступа к API. 

<B>Запуск тестов</B>

Для запуска всех тестов выполните команду:

bash

pytest -s

<B>Описание тестов</B>

-Проверка успешного создания и получения элементов.

-Проверка фильтрации, сортировки и пагинации списка.

-Проверка обновления элементов, включая негативные кейсы (обновление несуществующего элемента).

-Проверка удаления элементов и ошибок повторного удаления.

-Проверка корректной обработки запросов без токена с ожидаемыми кодами ошибок.

<B>Важно!</B>

Все ключевые параметры хранятся в config/constant.py для удобства настройки окружения.

Тесты используют pytest и requests.

Для расширенной отчетности можно подключить Allure.
